5.some

6.forEach

forEach是循环遍历数组中的每一个元素，等同于for循环。
forEach的callback支持3个参数：
currentValue：遍历到得当前元素
index：对应的数组索引
array：数组本身

forEach不会去遍历数组中空的元素。

array：数组本身
