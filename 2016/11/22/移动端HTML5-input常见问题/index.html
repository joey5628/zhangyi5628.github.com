3. input 输入框调出数字键盘

单独使用type=”number”时，iOS调起的并不是九宫格样式的数字键盘，如果需要调起九宫格的数字键盘需要加上 pattern=”[0-9]*” 属性

4. 搜索时，键盘的回车按钮文字设定为“搜索”

解决： input 使用 type=”search”，放在 form 表单内。两者结合就能使输入法中的回车按钮文字变为“搜索”
